Site Web de Reservation pour des Vacances en Famille - HTML/CSS/JavaSCript

Site créé pour les familles qui souhaitent planifier leurs prochaines vacances en famille. Page d'accueil montre un texte et une image de bienvenue accrocheurs et un petit paragraphe décrivant l'organisation. Page des attractions affichant toutes les attractions disponibles pour les enfants et pour la famille avec une animation JavaScript de diaporama parcourant les activités et affichant des informations plus détaillées. À manger page répertoriant les options de restauration disponibles sur des affiches de style carte avec des vignettes cliquables. Page de réservations avec type de chambre sélectionnable en cliquant sur l'image de la chambre souhaitée et un formulaire de réservation interactif validé avec JavaScipt. Le site contient également une option connexion et une option s'enregistrer avec une validation appropriée et des cookies locaux si vous créez un compte et vous connectez sur le même appareil. L'ensemble du site est bilingue avec un bouton EN/FR dans l'en-tête la langue du site Web de manière appropriée.

Application Android d'un Restaurant - Java/Gradle

La fenêtre d'acceuil est le menu alimentaire qui est divisé en fragments de style de carte horizontale déroulante pour les catégories d'aliments offerts par le restaurant et des fragments de texte verticaux déroulants pour les articles individuels. Cliquer sur une nouvelle catégorie change la liste des éléments. Toutes les catégories et tous les éléments sont des objets créés à partir d'une base de données et chargés dans leurs conteneurs respectifs. En cliquant sur un article, une nouvelle activité s'ouvre et affiche le nom complet, l'image, la description et le prix de l'article avec un sélecteur de quantité et un bouton pour ajouter au panier. L'application contient une barre de navigation accessible par un swipe ou en cliquant sur l'icône de menu de développement, et à travers elle, vous pouvez naviguer entre le Menu Alimentaire et les Commandes. Les commandes affichent tous les articles sélectionnés, le montant sélectionné, le prix cumulé et prix total.

Data Structures and Algorithms Projects - Java

Expérimentation avec de nombreuses structures de données et algorithmes différents, et leurs notations Big-O. Les types primitifs étant des éléments tels que les: booleans, characters, integers, floating-point numbers, et by-reference. Les types non primitifs étant des éléments tels que les arrays et les strings. Et très important, les types de données abstraits tels que les: arrays, lists, single- linked-lists, double-linked-lists, maps, sets, stacks, queues, priority-queues, heaps, trees, binary-trees, binary-search-trees, red-black trees, splay-trees, self-balancing -binary -trees. Et tout aussi important, les algorithmes de tri tels que les: heap-sort, insertion-sort, selection-sort, quick-sort, merge-sort and counting-sort.
